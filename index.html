Hazelcast Python Client¶

Python client implementation for Hazelcast, the open source in-memory data grid.

The New Documentation

This documentation site is for the 3.x versions of the client.

For the new documentation please visit hazelcast.readthedocs.io.

3.x¶

It is implemented using the Hazelcast Open Binary Client Protocol.

It supports Python 2.7, 3.4, 3.5, 3.6, 3.7, 3.8 and 3.9.
